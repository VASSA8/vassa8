## 👨‍💻 О себе

Высокомотивированный специалист, сочетающий технические навыки в веб-разработке с креативным подходом к визуальному контенту. В настоящее время фокусируюсь на создании эффективных фронтенд-решений и мобильных приложений.

* **🌐 Основное направление:** Frontend Development, Mobile Development (React Native)
* **💡 Ключевые технологии:** TypeScript, React, Vite, Node.js
* **🎯 Цель:** Разработка масштабируемых и поддерживаемых продуктов
